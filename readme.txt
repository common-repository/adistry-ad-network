== Description ==

Easily display Adistry ads anywhere on your Wordpress website using shortcodes.

[adistry adzone_id="10000"]

Notes:

*   Each adzone you create has it's own ID.  Access this ID from the "Get adzone embed code" page.
*   Replace the sample ID (10000) in the above shortcode with your adzone's id.
*   You may use the shortcode in any Blog or Page content, as well as the Text Widget (or anywhere else you set shortcodes to work)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `adistry/` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the above shortcode to any Blog, Page, or Text Widget

== Frequently Asked Questions ==

= What is the shortcode? =

Add this shortcode to any Blog, Page, or Text Widget (or anywhere else you set shortcodes to work)
